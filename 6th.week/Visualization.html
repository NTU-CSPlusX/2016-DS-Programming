R 的繪圖功能

  • 基本功能
  • ggplot2

基本功能

繪圖的要點

if (grepl("darwin", R.version$os)) par(family = "STKaiti", bg = "transparent")
plot(x = 1, y = 1, type = "p", xlab = "x座標軸", ylab = "y座標軸")

  • 繪圖的要點:
    • 完成主要的繪圖區域
      • 利用x 座標與y 座標定位點、線、面
    • 豐富其他說明文件
      • 例如：x 、y 座標軸與說明

基本功能的分類

  • 高級繪圖函數: plot、images、hist…
  • 低級繪圖函數: points、lines…

高級繪圖函數

  • 會重新建立圖形的函數
plot(x = 1, y = 1, type = "p", xlab = "x座標軸", ylab = "y座標軸")

低級繪圖函數

  • 會在原本的圖上作加註
plot(x = 1, y = 1, type = "p", xlab = "x座標軸", ylab = "y座標軸")
abline(v = 1)
abline(h = 1)
